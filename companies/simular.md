# Simular

> Former DeepMind scientists building autonomous AI agents that control your Mac and Windows PC — navigating interfaces, completing workflows, and operating software like a human knowledge worker.

## Company Overview

Simular is a research-driven startup building AI agents that can operate software on behalf of users by navigating interfaces, completing workflows, and interacting across applications. Unlike browser-only agents, Simular's approach encompasses the entire desktop operating system — controlling native applications, managing files, and executing multi-step workflows across the full computing environment.

Founded in 2024 by former DeepMind scientists specializing in reinforcement learning, the company released its 1.0 agent for Mac OS in December 2025 and was accepted into Microsoft's Windows 365 for Agents program, positioning it for cross-platform coverage.

## Product: Simular Agent

Simular's agent operates at the OS level, controlling the computer the way a human would:

- **Mac OS 1.0** released December 2025 — can navigate native Mac applications, manage files, and execute complex multi-step workflows
- **Windows support** in development through Microsoft's Windows 365 for Agents program (one of five companies accepted)
- Uses vision-based understanding of screen content combined with reinforcement learning to decide and execute actions

**Key differentiator:** While browser agents (Browser Use, OpenAI Operator) are limited to web interfaces, Simular operates across the full desktop — native apps, file systems, and system settings — making it capable of automating a broader range of knowledge work.

## Business Model

Simular is positioned as a desktop automation agent for knowledge workers. Likely monetization includes:

- Subscription-based access to the Simular agent
- Enterprise licensing for managed deployments
- Potential platform fees for third-party integrations

Specific pricing has not been publicly announced.

## What Makes Simular Interesting

1. **Full desktop control vs. browser-only:** Most agent startups focus on web browsing; Simular controls the entire OS — a more ambitious and potentially more valuable approach for automating real knowledge work.

2. **DeepMind RL expertise:** Founded by reinforcement learning specialists from DeepMind's Waymo team — the same RL techniques that powered AlphaGo and robotics are being applied to desktop automation.

3. **Microsoft partnership:** Acceptance into the Windows 365 for Agents program gives Simular direct collaboration with Microsoft on Windows agent development — a potential distribution advantage.

4. **Nvidia backing:** NVentures investment signals alignment with Nvidia's agent ecosystem strategy and potential access to optimized compute for agent inference.

5. **Cross-platform ambition:** Mac + Windows coverage would make Simular the most broadly deployed desktop agent, reaching virtually all knowledge workers.

## Uncertainties and Open Questions

- **Revenue and traction:** No public revenue or user numbers. Mac OS 1.0 only launched in December 2025.
- **Team size:** Small team; scaling will be essential to support enterprise customers and multi-platform development.
- **Reliability at scale:** Desktop automation is notoriously brittle — UI changes, permission dialogs, and edge cases make autonomous computer control challenging.
- **Competition:** Apple Intelligence, Microsoft Copilot, and Anthropic's computer use feature may absorb this capability into existing platforms.
- **Valuation:** Not publicly disclosed for either funding round.
