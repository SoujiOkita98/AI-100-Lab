# Qdrant

## Overview

Qdrant is an open-source vector search engine built in Rust, designed for production AI systems requiring fast, scalable similarity search over high-dimensional vectors. Founded in 2021 by Andre Zayarni and Andrey Vasnetsov in Berlin, Germany, the company provides composable vector search infrastructure that powers retrieval-augmented generation (RAG), recommendation systems, and semantic search applications. Qdrant raised a $50M Series B led by AVP in March 2026, bringing total funding to $88M. Investors include Spark Capital, Unusual Ventures, 42CAP, and Bosch Ventures.
