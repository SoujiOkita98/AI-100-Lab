# Trigger.dev

## Overview

Trigger.dev is a developer platform for building and running AI agents and workflows. It provides an SDK, deployment infrastructure, scaling, monitoring, and debugging tools so developers can create production-grade agents without managing infrastructure. The platform is used by more than 30,000 developers and handles hundreds of millions of AI agent executions per month. The open-source project has amassed over 12,000 GitHub stars.

## Founders & Key People

### Matt Aitken -- Co-Founder & CEO
- **Background:** Created two iPad apps that won Apple's iPad App of the Year award; previously built JSON Hero, an open-source JSON viewer used by 35K+ developers monthly; serial startup founder
- **Education:** B.A. in Architecture, University of Sheffield

### Eric Allam -- Co-Founder & CTO
- **Background:** Co-founded CodeSchool.com, an early code learning platform acquired in 2015; built JSON Hero alongside Aitken
- **Connection:** The four founders have worked together for 7+ years

### James Ritchie -- Co-Founder
- **Background:** Design and development agency director at Tiny Matter; MA in Communication, University of the Arts London

### Daniel Patel -- Co-Founder
- **Background:** Software engineering

## Products & Technology

1. **SDK:** TypeScript SDK for defining tasks, workflows, and AI agent logic
2. **Managed Infrastructure:** Automatic scaling, retries, timeouts, and fault-tolerant execution
3. **Monitoring & Debugging:** Real-time visibility into task execution, errors, and performance
4. **Cron & Scheduling:** Built-in support for scheduled and recurring tasks
5. **Durable Execution:** Reliable task execution with automatic retries and state persistence

## Business Model

- **Open-source with managed cloud:** Open-source core (12K+ GitHub stars); managed cloud platform for production use
- **Usage-based pricing:** Charges based on task executions
- **Target customers:** Developer teams building AI agents and workflows

## What Makes Trigger.dev Interesting

1. **Agentic AI infrastructure.** As AI agents become more prevalent, the need for reliable background task orchestration grows. Trigger.dev is building the plumbing layer that makes agents work reliably at scale.

2. **Proven at scale.** Hundreds of millions of executions per month and 30,000+ developers demonstrate real production traction, not just developer interest.

3. **Strong founding team chemistry.** Four co-founders who have worked together for 7+ years, with a track record of building successful developer tools (CodeSchool, JSON Hero), reduces execution risk.

4. **UK-based YC success story.** One of the notable examples of a London-based team succeeding in YC and scaling a developer tools company globally.
