# Moda

## Overview

Moda provides product analytics specifically designed for AI-powered products. Traditional analytics tools (Amplitude, Mixpanel) were built for deterministic software with defined user flows. AI-powered products are fundamentally different -- outputs are non-deterministic, user interactions are open-ended, and success metrics are harder to define. Moda fills this gap by providing analytics infrastructure that helps product teams understand how users interact with AI features, track AI output quality, and optimize AI-powered experiences.
