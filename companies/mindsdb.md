# MindsDB

## Overview

MindsDB is an open-source AI platform that enables developers to integrate machine learning into applications using familiar SQL syntax. Founded in 2017 by Jorge Torres and Adam Carrigan, inspired by Iain M. Banks's Culture series, the company connects AI models directly to data sources, making ML accessible without requiring specialized expertise. MindsDB is backed by $50M+ in funding from Benchmark, Mayfield, Y Combinator, and NVIDIA's NVentures. The platform supports integration with hundreds of data sources and AI models.
