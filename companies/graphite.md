# Graphite

## Overview

Graphite is an AI-powered code review and developer workflow platform that leverages models from Anthropic and OpenAI to flag errors, suggest improvements, and accelerate the review process. Originally founded as a mobile development tooling company, Graphite pivoted to code review early on and became one of the fastest-growing developer tools startups. In December 2025, Graphite was acquired by Cursor (Anysphere), combining Graphite's code review capabilities with Cursor's AI coding editor.

## Founders and Leadership

- **Background**: Former software engineer at Facebook. Experience building developer-facing products at scale.

### Merrill Lutsky -- Co-Founder

## Product and Technology

### Core Platform

- **AI Code Review**: Uses Anthropic's and OpenAI's models to review pull requests, flag errors, identify potential bugs, and suggest improvements
- **Diamond**: Graphite's AI code review product launched alongside the Series B announcement
- **Stacking Workflow**: Graphite's original value proposition was "stacked diffs" (stacked pull requests), allowing developers to work on multiple dependent changes simultaneously rather than waiting for sequential reviews
- **GitHub Integration**: Deep integration with GitHub as the primary code hosting platform

### Key Metrics (pre-acquisition)

- **Revenue grew 20x in 2024**
- **500+ companies** using the platform, including Shopify, Snowflake, Figma, and Perplexity
- **Tens of thousands of engineers** on the platform

## Acquisition by Cursor (December 2025)

In December 2025, Cursor (Anysphere) acquired Graphite, combining AI code generation (Cursor) with AI code review (Graphite) into an integrated developer workflow. This acquisition represents a strategic consolidation in the AI developer tools market.

## What Makes Graphite Interesting (Research Notes)

1. **The Cursor acquisition validates the "review layer" thesis.** Cursor, the leading AI code editor, acquiring Graphite confirms that code review is a critical complement to code generation. The combined product could offer end-to-end AI-powered development.

2. **Anthropic's investment is notable.** Having the maker of Claude -- one of the most capable coding LLMs -- as a strategic investor gives Graphite privileged access to frontier model capabilities.

3. **20x revenue growth in 2024** is exceptional and suggests strong product-market fit during a period of rapid AI coding tool adoption.

4. **The customer roster (Shopify, Snowflake, Figma, Perplexity)** demonstrates enterprise credibility and suggests that AI code review is moving from experimental to production-grade in top engineering organizations.

5. **Pivot from mobile tooling to code review** was a smart early bet that positioned Graphite ahead of the AI coding wave.
