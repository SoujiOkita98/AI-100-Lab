# Kerno

## Overview

Kerno is a Dublin, Ireland-based startup founded in 2022 that builds an AI-powered validation engine for backend development. The platform automatically tests and validates code changes against real dependencies inside the developer's IDE before a pull request is raised. As AI coding assistants like Cursor and Claude Code become central to development workflows, Kerno positions itself as the validation layer that ensures AI-generated code actually works. The company raised EUR 1.69M (~$1.86M) in seed funding.

## Product

Kerno automates integration testing end-to-end inside the IDE:
- Understands codebase and generates meaningful tests
- Spins up test environments with real dependencies
- Captures baseline behavior of REST endpoint critical paths
- Compares code changes against baselines and flags diffs
- Auto-updates baseline tests as the application evolves
- Uses eBPF and application graphRAG technology for precision context

**Security:** Data encrypted in-transit (TLS 1.2) and at-rest (AES 256-bit). Private code is never stored or used for training.
