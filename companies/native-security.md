# Native

## Overview

Native is a cybersecurity startup building a platform for managing cloud security policies across multi-cloud environments. Founded by AWS veterans, the company uses AI to automate security policy creation and enforcement, addressing the growing complexity of securing workloads spread across AWS, Azure, GCP, and other cloud providers.

## Funding Story

The $31M Series A led by Ballistic Ventures is a solid raise for a cloud security startup from AWS veterans. Ballistic Ventures specializes exclusively in cybersecurity investments.

## What Makes Them Interesting

1. **AWS insider knowledge.** Founders who built AWS security understand where the gaps are better than anyone.
2. **Multi-cloud reality.** As enterprises run workloads across 3-4 cloud providers, unified security policy management is a growing pain point.
3. **AI-driven policy automation.** Using AI to generate and maintain security policies reduces the manual toil that currently overwhelms security teams.
