# Continue

## Overview

Continue is the leading open-source AI code assistant, offering IDE extensions for VS Code and JetBrains that connect to any LLM (OpenAI, Anthropic, local models, etc.) and let developers add custom context from platforms like Jira, Confluence, and internal documentation. Unlike proprietary alternatives like Cursor or GitHub Copilot, Continue gives developers and enterprises full control over which models they use, where their code is sent, and how the assistant behaves. With 23,000+ GitHub stars and 11,000+ Discord community members, Continue has built a significant open-source community.

## Product and Technology

### Open-Source IDE Extensions (v1.0, Feb 2025)

Continue's core product is a pair of open-source IDE extensions for VS Code and JetBrains that provide:

- **Autocomplete**: Inline code suggestions from any LLM
- **Chat**: Conversational coding assistant with codebase context
- **Edit**: Natural-language-driven code modifications
- **Agent**: Deeper codebase interactions and multi-step workflows

### Model Flexibility

A key differentiator: Continue connects to any LLM -- OpenAI, Anthropic, Google, Ollama (local models), and others. This gives enterprises the ability to use their preferred model provider, run models locally for sensitive code, or switch providers without changing workflows.

### Continue Hub (Feb 2025)

A registry for developers to create, share, and discover custom AI coding assistants. Analogous to Docker Hub or Hugging Face Hub -- a marketplace for pre-configured assistant configurations, context providers, and custom tools.

### Custom Context

Teams can pull in context from Jira, Confluence, internal docs, and other data sources, making the AI assistant aware of project-specific knowledge beyond just the code.

### Continue CLI and CI Integration (2026)

Continue has evolved to also offer source-controlled AI checks enforceable in CI pipelines, positioning itself as "quality control for your software factory."

## What Makes Continue Interesting

1. **The open-source alternative to Cursor and Copilot.** In a market dominated by proprietary AI coding tools, Continue offers the only serious open-source alternative with broad model support. This matters for enterprises with data sovereignty requirements, air-gapped environments, or a preference for local model inference.

2. **23,000+ GitHub stars demonstrate genuine developer demand.** Open-source traction at this scale -- with minimal funding -- suggests strong product-market fit in the developer community.

3. **Model-agnostic architecture is strategically wise.** As LLM leadership shifts rapidly between providers, Continue's ability to plug into any model means it benefits from LLM improvements without being locked to a single provider. This is a hedge that proprietary tools cannot easily replicate.

4. **Continue Hub could become the "Hugging Face for coding assistants."** If the hub gains traction as a registry for custom AI coding configurations, it creates a network effect and community moat.

5. **Capital efficiency is extreme.** ~$5.6M in total funding, ~10 employees, 23,000+ GitHub stars, and a V1.0 product launch -- this is remarkably lean. The question is whether the team can scale to capture the enterprise opportunity.

6. **Y Combinator and Heavybit are strong developer-tools investors.** Heavybit, specifically, has a strong track record backing developer infrastructure companies (LaunchDarkly, Snyk, CircleCI).
