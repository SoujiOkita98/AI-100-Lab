# Exa

## Overview

Exa (formerly Metaphor) is an AI-native search engine built specifically for machine consumption rather than human browsing. The company provides a search API and data retrieval layer that allows large language models, AI agents, and developer applications to find and access high-quality web content. Unlike traditional search engines optimized for ad-driven human click-through, Exa is designed to return clean, structured, high-fidelity information optimized for AI pipelines.

## Founding Story and Team Origins

Exa was founded in 2021 by Will Bryk (CEO) and Jeff Wang, who were classmates at Harvard. Dan McArdle is also listed as a co-founder. The company was part of Y Combinator and built its search engine from scratch using neural/embedding-based retrieval rather than traditional keyword matching.

## Business Model

- **Search API:** Developer-facing API that allows AI applications to search and retrieve web content programmatically
- **Websets:** A high-compute search product for comprehensive, structured queries (e.g., "find all AI startups in the US that raised funding in 2024-2025")
- **Usage-based pricing:** Charges per API call / query volume
- **Enterprise customers:** Serves major companies including AWS, Databricks, Vercel, and Anysphere (Cursor)

Revenue growth has been explosive: estimated ~$10M ARR by September 2025, up ~1,010% YoY from ~$0.9M in September 2024 (Sacra estimates).

## Key Differentiators

- Purpose-built for AI/LLM consumption (not human browsing)
- Neural/embedding-based search rather than keyword matching
- Clean, structured data output optimized for RAG (retrieval-augmented generation) pipelines
- Explosive revenue growth (~10x YoY)
- Strategic position as the "information retrieval layer" for the AI agent ecosystem
- Customer base includes major AI/developer platforms
