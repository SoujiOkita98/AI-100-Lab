# Tensol

## Overview

Tensol builds orchestration infrastructure for multi-agent AI systems. As AI applications evolve from single agents to complex multi-agent architectures where specialized agents collaborate on tasks, Tensol provides the coordination layer that manages agent communication, task delegation, error handling, and state management. This addresses a key infrastructure gap: while tools like LangChain and CrewAI help build individual agents, production multi-agent systems require robust orchestration.
