# Sourcegraph

## Overview

Sourcegraph is the company behind the industry-leading **code search** platform used by large engineering organizations to navigate massive, multi-repository codebases. In 2023, Sourcegraph launched **Cody**, an AI coding assistant that leverages its deep code graph and search infrastructure to provide context-aware code generation, explanation, and editing. In 2025, Sourcegraph launched **Amp**, a full AI-native coding agent, and subsequently announced that Sourcegraph and Amp would become two independent companies, with the co-founders leading the new Amp Inc.

## Product and Technology

### Code Search (Core Product)

Sourcegraph's original product enables universal code search across all repositories, branches, and code hosts. It supports regex, structural, and symbol-based search across millions of files. Adopted by large enterprises with complex, multi-repo environments.

### Cody (AI Coding Assistant, launched 2023)

Cody is an AI coding assistant that uses Sourcegraph's code graph as context to provide more accurate code generation, explanation, and editing. Key differentiator: Cody indexes entire codebases for retrieval-augmented generation, reducing hallucinations compared to assistants with limited context windows.

Note: In 2025, Sourcegraph discontinued the Cody Free, Pro, and Enterprise Starter plans, consolidating around enterprise offerings.

### Amp (AI Coding Agent, launched 2025)

Amp is a full AI-native coding environment and coding agent. It represents Sourcegraph's bet on agentic AI for software development. Amp is being spun out as an independent company (Amp Inc.) led by co-founders Quinn Slack and Beyang Liu.

## Business Model

- **Code Search**: Enterprise SaaS, per-seat pricing for large engineering teams.
- **Cody/Amp**: Shifting to enterprise-focused tiers after discontinuing free and lower-tier plans.
- **Target Customer**: Large engineering organizations with complex, multi-repository codebases.

## What Makes Sourcegraph Interesting

1. **Deep code intelligence moat.** Sourcegraph has spent over a decade building the infrastructure to index, search, and understand code at scale. This code graph is a genuine technical moat that competitors cannot easily replicate, and it provides superior context for AI features.

2. **The Amp spin-off is a bold strategic move.** Splitting into two independent companies -- one for code search, one for agentic AI coding -- reflects a belief that these are distinct markets requiring distinct focus. It also allows Amp to move fast without being constrained by Sourcegraph's enterprise sales cycles.

3. **Revenue trajectory suggests enterprise traction but not breakout growth.** $50M in revenue at a $2.6B valuation (from 2021) implies the company has not grown into its valuation as fast as hoped. The AI pivot via Cody and Amp is an attempt to accelerate growth.

4. **Organizational turbulence.** Multiple reports of layoffs and restructuring ("opt-in severance" rounds) suggest the company has gone through significant internal changes. The spin-off may be partly a response to these challenges.

5. **Competing on context depth, not just speed.** While Cursor and Copilot optimize for individual developer speed, Sourcegraph's differentiator is understanding entire codebases -- a harder problem that matters more at enterprise scale.

## Key Uncertainties

- Whether the $2.6B valuation from 2021 will hold in subsequent rounds, given the spin-off and layoffs
- How the Sourcegraph/Amp split will affect enterprise customers and revenue allocation
- Whether Amp can compete effectively with Cursor, Copilot, and other agentic coding tools as a standalone company
- Exact current employee count and revenue split between the two entities post-split
