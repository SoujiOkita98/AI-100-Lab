# 21st (21st.dev)

## Overview

21st.dev is the largest React component registry for AI applications, serving 1.4 million developers and 200K monthly active users. The platform provides building blocks for creating AI agents and agent-powered apps, including UI components, agent templates, and an SDK with built-in UI, tools, memory, observability, and managed infrastructure. Founded as an orchestration layer for coding agents, 21st.dev sits at the intersection of the agentic AI wave and frontend development infrastructure.
