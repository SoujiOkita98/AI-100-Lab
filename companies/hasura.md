# Hasura -- AI Data Infrastructure & API Platform

## Company Overview

Hasura is a developer tooling company that provides instant data access and API generation for enterprises. Originally known for its open-source GraphQL engine that auto-generates APIs from databases, Hasura has evolved into an AI data integration platform. Its newer product, PromptQL, provides a natural-language query interface for AI applications to access structured and unstructured enterprise data.

Hasura's core thesis is that AI applications need reliable, secure, and performant access to enterprise data -- and that building this data access layer from scratch is one of the biggest bottlenecks in AI deployment. The open-source Hasura GraphQL Engine has been downloaded over 400 million times.

## What Makes Hasura Interesting

1. **IIT Madras to open-source unicorn:** Tanmai Gopal built one of the most widely adopted open-source developer tools (400M+ downloads) to a $1B valuation.
2. **Pivoting to AI data infrastructure:** Evolution from GraphQL API engine to AI data access platform (PromptQL) positions Hasura in a critical infrastructure layer.
3. **Indian founder in developer tools:** Operating in a space more commonly associated with US-based founders, broadening the picture of where Indian founders compete.
4. **Open-source community as moat:** 400M+ downloads creates distribution and feedback loops difficult to replicate.
5. **Both founders are Indian:** Unlike many companies where the Indian founder is one of several, both Hasura co-founders are Indian.
