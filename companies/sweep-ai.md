# Sweep

## Overview

Sweep is an AI coding assistant initially launched as a bot that turns bugs and feature requests into pull requests with code. The company has since pivoted to become an AI coding assistant for JetBrains IDEs, competing in the rapidly growing AI-assisted development space. Sweep emerged from Y Combinator's S23 batch and was one of the early companies building AI-powered developer tools before the category exploded.

## Founders & Key People

### William Zeng -- Co-Founder
- **Background:** Was the youngest senior machine learning engineer at Roblox, where he gained deep experience in ML systems and AI-powered code generation
- **Expertise:** Machine learning, large language models, code generation

### Kevin Lu -- Co-Founder
- **Background:** Top student in his class at University of Waterloo in Math/CS; strong academic foundation in AI and computer science
- **Expertise:** Mathematics, computer science, AI systems

## Products & Technology

1. **JetBrains AI Assistant:** AI-powered coding assistance integrated directly into JetBrains IDEs (IntelliJ, PyCharm, WebStorm, etc.)
2. **Code Generation:** Automated code writing from natural language descriptions
3. **Bug Fix Automation:** AI-driven identification and fixing of code issues

The company initially launched as a GitHub bot that automatically created pull requests for reported bugs and feature requests, leveraging LLMs to understand codebases and generate appropriate code changes. The pivot to JetBrains reflects a strategic decision to focus on the IDE experience.

## Business Model

- **Product-led growth:** Developer tool sold directly to individual developers and teams
- **Target market:** Developers using JetBrains IDEs (millions of users globally)

## What Makes Sweep Interesting

1. **Early mover in AI coding tools.** Sweep was one of the first YC-backed startups to tackle AI-automated code generation, launching before the category became crowded with competitors like Cursor, Cody, and others.

2. **Strategic pivot.** The shift from a GitHub bot to a JetBrains-integrated assistant shows adaptability -- recognizing that developers want AI assistance embedded in their IDE workflow, not as a separate service.

3. **Replit connections.** Having the CEO and VP of AI of Replit as angel investors provides both credibility in the AI coding space and access to deep technical expertise.

4. **Open-source roots.** The Sweep GitHub repository has been an important part of the company's developer relations and community building.
