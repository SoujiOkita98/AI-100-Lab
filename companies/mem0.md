# Mem0

## Overview

Mem0 is the memory layer for AI applications and agents, providing persistent, adaptive memory that enables personalized AI experiences. Founded in 2023 by Taranjeet Singh (ex-Khatabook, Y Combinator S24) and Deshraj Yadav (ex-Tesla Autopilot AI Platform), the open-source project has surpassed 41,000 GitHub stars and recorded over 13 million Python package downloads. API calls grew from 35 million in Q1 2025 to 186 million in Q3. The company raised a $24M Series A led by Basis Set Ventures with participation from Peak XV Partners, GitHub Fund, and Y Combinator.
