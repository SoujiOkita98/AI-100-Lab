# Factory

## Overview

Factory is building an "agent-native" software development platform centered on autonomous AI agents called "Droids" that handle full tasks across the software development lifecycle. Unlike AI coding assistants that augment developers with suggestions, Factory's Droids are designed to independently complete engineering tasks -- from writing code to running tests to deploying changes -- while human engineers focus on high-level architectural decisions and product direction.

## Product and Technology

### Droids (Core Product)

- **Autonomous Software Agents**: Droids handle full development tasks end-to-end rather than offering code suggestions
- **Agent-Native Development**: A development model where AI agents are first-class participants in the engineering workflow, not just assistants
- **Benchmark Performance**: Factory's Droids ranked No. 1 on Terminal Bench, the industry benchmark also used by tools like Claude Code and Cursor
- **Task Types**: Bug fixes, feature implementation, code refactoring, test writing, and deployment tasks

### Development Philosophy

Factory's model represents a paradigm shift from "AI-assisted development" (human writes code with AI help) to "agent-native development" (AI agents write code with human oversight). This positions them at the more ambitious end of the AI coding spectrum.

## What Makes Factory Interesting (Research Notes)

1. **Top benchmark performance.** Ranking No. 1 on Terminal Bench validates that Factory's agents can genuinely complete real-world software engineering tasks, not just toy examples.

2. **"Agent-native" is a bold bet.** Most successful AI coding tools (Cursor, Copilot) augment developers rather than replacing tasks entirely. Factory is betting that autonomous agents will handle complete tasks, which is higher risk but higher reward if it works.

3. **Investor quality is exceptional.** Sequoia, NEA, NVIDIA, and J.P. Morgan in the cap table signals strong institutional conviction. Sequoia leading the Series A at a relatively early stage is particularly notable.

4. **$300M valuation at Series B** reflects rapid value creation from the $120M Series A valuation, driven by strong product-market signal from Droids' benchmark performance and enterprise interest.

5. **Competition with Cognition AI (Devin)** is direct -- both are building autonomous software development agents. Factory's benchmark leadership may give it an edge in enterprise adoption.

## Key Uncertainties

- Revenue and customer metrics are not publicly disclosed
- Whether autonomous agents can reliably handle complex, real-world codebases (benchmarks vs. production performance gap)
- The exact team size and composition is not well documented
- How enterprises will adopt "agent-native" workflows alongside existing human engineering teams
