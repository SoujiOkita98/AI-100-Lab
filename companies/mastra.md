# Mastra

## Overview

Mastra is an open-source, modular TypeScript framework for building AI-powered applications and agents. The founding team spent the last decade building Gatsby, the React-based framework used by hundreds of thousands of developers. Mastra is now the third-fastest-growing JavaScript framework ever measured, reaching 300,000+ weekly npm downloads, 19,400+ GitHub stars, and production use at companies like Replit and WorkOS.
