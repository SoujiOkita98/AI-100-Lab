# Data Gathering Principles

1. **Quality over completeness** — An empty field is better than a wrong one.
2. **Every claim is traceable** — Include the source URL. If there's no source, flag it.
3. **Accuracy over speed** — Mark uncertainty explicitly (e.g. "estimated", "unconfirmed"). Never guess silently.
4. **Schema is a guide, not a cage** — If something interesting doesn't fit a field, capture it anyway. If a field isn't applicable, skip it.
5. **Sectors are fluid** — Use the taxonomy as a starting point, add new ones as needed.
6. **Founder origin is free text** — Capture the nuance (e.g. "born in Beijing, PhD at MIT, stayed in US" vs just "mainland-chinese"). The roll-up field is for quick filtering.
7. **Funding rounds are sourced individually** — Each round should link to the announcement/article.
